# Pixel Chomp Level Design Skill

## Purpose
Design mazes that create clear routing decisions, tension, escape opportunities, and power-up risk/reward.

## Principles
- readability first
- no dead visual clutter
- multiple route choices
- intentional choke points
- fair recovery paths after respawn

## Maze checklist
- clear outer loop and inner pressure lanes
- at least a few safe-ish rotation routes
- power pellets placed where they create decisions, not freebies
- tunnels used for escape, not abuse
- spawn zone placement does not trap the player unfairly

## Layout guidance
- start with one symmetrical or near-symmetrical maze for easier tuning
- create long lanes for chase pressure
- create junction clusters for mind games
- avoid too many tiny micro-corridors
- keep pellet placement dense enough to encourage route planning

## Expansion path
After MVP:
- maze 2 can add more asymmetry
- maze 3 can add hazard tiles or moving gates
- challenge mazes can change tunnel behavior
